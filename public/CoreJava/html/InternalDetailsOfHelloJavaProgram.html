5. Internal Details of Hello Java Program
In the previous section, we have created Java Hello World program and learn how to
compile and run a Java program. In this section, we are going to learn, what happens while we
compile and run the Java program. Moreover, we will see some questions based on the first
program.
What happens at compile time?
At compile time, the Java file is compiled by Java Compiler (It does not interact with
OS) and converts the Java code into bytecode.
What happens at runtime?
At runtime, the following steps are performed:
